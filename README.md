# 계산기

## 지원 기능

팀원들이 기능을 한 줄씩 작성할 수 있도록 아래 표를 사용하세요.

| 기능 | 설명 |
| --- | --- |

| add (덧셈) | 
두 숫자를 더한 결과를 반환합니다. | 
+ 숫자가 아닌 값은 '메롱'으로 반환합니다 |
+ 다른 기능 추가되면 이런식으로 표시 가능

| 예: 뺄셈 | 두 숫자의 차를 반환합니다. |

| 예: 곱셈 | 두 숫자의 곱을 반환합니다. |

| 예: 나눗셈 | 두 숫자를 나눈 결과를 반환합니다. |

| 예: 거듭제곱 | 한 숫자의 거듭 제곱 결과를 반환합니다. |
